# InterfaceWorks - Prerelease builds

> This branch is made for development builds with code that might not fully work. Use at Your own risk!

## What is it?
InterfaceWorks is a simple UI State controller for Roblox.

## Docs
All documentation is available [here](https://sratique.github.io/InterfaceWorks-Docs/).

## Tooling used
To build a binary Yourself, You will need:
- [Rojo](https://rojo.space/)
- [darklua](https://darklua.com/)

All the build properties files are in the repo.

## Disclaimer
This is **NOT** meant to replace Roblox React, at least as of right now.
> This is a solo dev project so please be patient for any bug fixes or feature updates.
